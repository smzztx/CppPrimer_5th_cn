## 练习1.6

> 解释下面程序片段是否合法。
```cpp
std::cout << "The sum of " << v1;
      << " and " << v2;
      << " is " << v1 + v2 << std::endl;
```
如果程序是合法的，它的输出是什么？如果程序不合法，原因何在？应该如何修正？
  
非法，前两行的末尾有分号，表示语句结束，第2、3两行为两条新的语句。而这两条语句在 “<<” 之前缺少了输出流，应在 “<<” 之前加上 “std::cout” ，或者去掉前两行末尾的分号。  
我这里是去掉前两行末尾的分号，修改后的代码如下：
```cpp
std::cout << "The sum of " << v1
          << " and " << v2
          << " is " << v1 + v2 << std::endl;
```

        std::cout << i << std::endl;

Ex1.11:
```cpp
#include <iostream>

int main()
{
    std::cout << "please input two integers:\n";
    int small = 0, big = 0;
    std::cin >> small >> big;

    if (small > big)
    {
        int tmp = small;
        small = big;
        big = tmp;
    }

    for (int i = small; i != big; ++i)
        std::cout << i << std::endl;

    return 0;
}
```
